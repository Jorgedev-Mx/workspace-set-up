## 📚 Requisitos Previos

Antes de comenzar asegúrate de tener instalados los siguientes componentes:

- Sistema operativo: **Windows 10 / 11**
- NeoVim **0.9 o superior**
- Node.js **18+**
- PHP **8.x** y **Composer**
- Git
- Administrador de paquetes para Windows (opcional): **Chocolatey** o **Scoop**
